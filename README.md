# test_books
Тестовое задание: мини‑сервис “Reading List” 
(Python / FastAPI / ORM) 
## Описание
    
Коротко о задаче
Сделать небольшой REST API сервис «Reading List»:
пользователи ведут списки материалов (книги/статьи)
с метками и статусами.
Нужны простые CRUD‑операции и базовая фильтрация.
## Как запустить
Скачайте репозиторий
```commandline
git clone https://github.com/AlbertSabirzianov/test_books.git
```
Перейдите в каталог проекта
```commandline
cd test_books
```
Создайте виртуальное окружение
```commandline
python -m venv venv
```
Активируйте окружение
```commandline
source venv/Scripts/activate
```
Установите зависимости
```commandline
pip install -r requirements.txt
```
Перейдите в каталог с кодом
```commandline
cd src
```
Создайте файл .env с переменными окружения в этой директории
```text
PORT=8000
HOST="127.0.0.1"
```
Запустите приложение
```commandline
python main.py
```
Документация будет доступна
по адресу [http://{HOST}:{PORT}/docs#/](ef)

## Seed Данные
Для того чтобы быстро загрузить seed данные в систему
просто воспользуйтесь эндпоинтом /test, он быстро заполнит систему 
тестовыми данными.
## Упрощения
В данном приложении есть все необходимые CRUD‑операции и базовая 
фильтрация, но нет таких вещей как
- авторизация
- права пользователя
## База данных
В этом проекте используется SQLite в связке с SqlAlchemy
после запуска приложения в директории src появится файл base.db
который детально можно посмотреть например с помощью DB Browser (Sqlite)
